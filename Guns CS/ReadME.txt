El archivo config es para elegir las armas dentro de hay todas las categorias

Ejemplo:
Dentro del archivo config.py
b42--Elige la categoria numero 4 para elegir 2--AK-47
La letra "b" es la que abrira la tienda,puedes cambiarla dependiendo de la letra que uses para abrir la tienda

Hay 3 configuraciones las cuales puedes cambiar por cualquier arma o herramienta

El delay es el tiempo que tardara en comprar las armas

Delay 350 milisegundos
